# The Cookbook

A personal recipe journal I built for my own kitchen. Private, installable, and designed to actually feel good to use.

---

## What it does

**Collect your recipes** — Write them from scratch or paste in a JSON and let the form fill itself. Add ingredients, steps, tips, photos, nutrition — everything in one place. There's also an AI prompt included that extracts any recipe into the right JSON format, ready to import.

**Cook with it** — Tap into cook mode for a clean, step-by-step view with type big enough to read across the counter. Adjust servings on the fly and the ingredient amounts update automatically.

**Browse and search** — A visual grid with live search and category filters. Fast to scan, pleasant to look at.

**Install it** — Works as a PWA. Add it to your home screen and it opens like a native app, no browser chrome.

**Looks the part** — Smooth page transitions, entrance animations, a dark mode that's actually dark. Designed to feel more like a cookbook than a web app.

**Private by default** — Login required. Nobody else gets in.
